# Otto's Personal Portfolio Website

## Overview
This is an ongoing personal website/portfolio project. There is no fixed scope — features and content will be added incrementally over time.

## Tech Stack
- **Framework:** React (JavaScript)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4 (via `@tailwindcss/vite` plugin)
- **Package Manager:** npm

## Project Structure
```
ottomaportfolio/
├── index.html            # Entry HTML
├── vite.config.js        # Vite config (React + Tailwind plugins)
├── package.json
├── .gitignore
├── src/
│   ├── main.jsx          # React entry point
│   ├── App.jsx           # Root component
│   ├── App.css           # Component styles (currently empty)
│   ├── index.css          # Tailwind entry (@import "tailwindcss")
│   └── assets/           # Static assets (images, SVGs, etc.)
└── public/               # Public static files
```

## Current State
- Basic Vite + React + Tailwind project scaffolded
- Default Vite template content has been cleaned out
- App.jsx contains a minimal "Hello, World!" placeholder
- Dev server runs on `http://localhost:5173/`

## Commands
- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run preview` — Preview production build
